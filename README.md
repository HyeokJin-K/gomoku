

## 오목 Team Project 
- 담당: 오목 AI 담당


## 주요 기술
- Minimax + Alpha-Beta Pruning AI 알고리즘
  
- 난이도별 탐색 깊이 및 후보수 조절
  - Easy: 후보 2개, depth 7
  - Normal: 후보 4개, depth 7  
  - Hard: 후보 6개, depth 7
    
- 그리디 알고리즘 기반 후보수 선별 최적화

- 비동기 처리

- 휴리스틱 평가 함수

## 주요 클래스
- 'GameResultChecker.cs': 게임 승패 판정 계산
- 'GomokuAI.cs': 알고리즘 기반 오목AI
